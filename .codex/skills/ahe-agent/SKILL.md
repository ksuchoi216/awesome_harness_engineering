# AHE Agent

Use this skill when the user invokes `$ahe-agent`.

## Command Workflow: ahe-agent

- Read `AGENTS.md`.
- Ask for the new project purpose when the current purpose is missing or outdated.
- Modify only the `PROJECT_PURPOSE` portion of `AGENTS.md`.
- Do not rewrite other sections of `AGENTS.md`.
