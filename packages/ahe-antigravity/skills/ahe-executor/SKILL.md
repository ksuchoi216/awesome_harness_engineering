# AHE Executor

This skill executes engineering plans generated by Awesome Harness Engineering.

## Instructions
When the user asks to implement or execute a plan, check the `.plans/` directory for the corresponding plan file (e.g. `.plans/{plan_name}.md`). Read the plan file to understand the required changes and follow it step-by-step.
