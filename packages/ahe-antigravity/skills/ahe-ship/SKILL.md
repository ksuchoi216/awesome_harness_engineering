# AHE Executor

This skill executes engineering plans generated by Awesome Harness Engineering.

## Instructions
When the user asks to implement or execute a plan, check the `.plans/` directory for the corresponding plan file (e.g. `.plans/{plan_name}.md`). Read the plan file, execute every listed requirement, and omit nothing.

Do not repeat, rewrite, summarize, or restate the plan.

Do not print the plan contents back to the user.

Start doing the work immediately. Only ask a question when execution is truly blocked by missing required information that cannot be derived from the repo or plan.

The Antigravity CLI runs this execution with the model `Gemini 3.1 Pro (High)`.

Print the exact line `AHE_PLAN_COMPLETE` only when the full plan is finished with no skipped work, no unresolved blocker, and no omitted verification.

Do not print `AHE_PLAN_COMPLETE` when any part of the plan is incomplete, deferred, blocked, or only partially verified.
